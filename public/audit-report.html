Что требует правки файлов (НЕ из админки)

Для изменения нижеследующего нужно редактировать файлы messages/cs.json, messages/en.json, messages/ru.json и делать rebuild + deploy:

  • Бронирование: все UI-тексты (лейблы полей, кнопки, GDPR, подсказки)
  • Страница "Pro firmy" полностью
  • Юридические страницы (Obchodní podmínky, Zásady ochrany osobních údajů)
  • Навигация хедера
  • SEO метаданные (title/description для каждой страницы)
  • Категории дополнительных услуг ("Péče o interiér", "Ochrana a vosky")
  • Общие слова: "Kč", "od", "Načítání...", "Zpět", "Pokračovat" и т.д.
